# Learn OpenGL

A private repository where I keep the code I develop while learning OpenGL. I am following the tutorial at (Learn OpenGL)[https://learnopengl.com/] by Joey de Vries. The code is not one-to-one identical to the tutorial; I learn the concepts and apply my own optimizations wherever needed. In the future, I will add some CMake files and a dependencies manifest once I learn how to use CMake and vcpkg.
